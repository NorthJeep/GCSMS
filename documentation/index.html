Overview

Bucket Admin is a premium admin dashboard template with flat design concept. Flat color, Customized Chart, Easy to customize and developer friendly code. It is fully responsive admin dashboard template built with Bootstrap 3+ Framework, HTML5 and CSS3, Media query, touch friendly and very much adaptive with any size viewport including iPhone, iPad, android phone and tablet. It has a huge collection of reusable UI components and integrated with latest jQuery plugins. It can be used for all type of web applications like custom admin panel, project management system, admin dashboard, application backend, CMS, CRM. A large number of widget are included here to make you

Powered by jQuery and Bootstrap, Bucket Admin includes many plugins implemented in 41 separate HTML files and also has a large selection of Iconic fonts in different Styles along with a wide array of Widgets, Charts, Notifications, Tooltips. It is a complete admin panel for any kind of web presence.

Font

BucketAdmin uses Open Sans and Lato font from google fonts. The font imported in main css file: css/style.css 


/* Import fonts */
@import url(http://fonts.googleapis.com/css?family=Open+Sans:400,300,300italic,400italic,600,600italic,700,700italic,800,800italic);
@import url(http://fonts.googleapis.com/css?family=Lato:400,100,100italic,300,300italic,400italic,700,700italic,900,900italic);
